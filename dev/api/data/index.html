Data Structures

Pool

MichiBoost.Pool — Type
Pool

Data container for training and prediction.

Stores numerical features as a Float64 matrix, categorical features as integer-encoded vectors, labels, sample weights, and feature metadata. String columns are automatically detected as categorical.

Construct via the Pool constructor with keyword arguments for label, cat_features, weight, etc.

Fields

  • n_samples::Int — number of rows
  • n_features::Int — total number of columns (numerical + categorical)
  • label — target vector (Float64), or nothing
  • weight — sample weights, or nothing
  • feature_names — column names as Symbols
MichiBoost.slice — Function
slice(pool::Pool, indices) -> Pool

Return a new Pool containing only the rows at indices.

subset = slice(pool, 1:100)

The Pool constructor accepts a text_features keyword argument (same format as cat_features) that is treated identically as categorical features.
