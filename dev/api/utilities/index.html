Utilities

Feature Importance

MichiBoost.feature_importance — Function
feature_importance(model::MichiBoostModel) -> Vector{Pair{Symbol, Float64}}

Compute feature importances based on split frequency across all trees.

Each pair is feature_name => percentage, sorted descending. Percentages sum to 100. Features never used in any split get 0.

feature_importance(model) -> Vector{Pair{Symbol, Float64}}

Return feature importances as feature_name => percentage pairs, sorted by importance (descending). Importance is based on how often each feature was chosen for a split across all trees.

Example

fi = feature_importance(model)
# [:num_1 => 62.5, :num_3 => 25.0, :cat_1 => 12.5]

SHAP Feature Attributions

MichiBoost.shap_values — Function
shap_values(model::MichiBoostModel, pool::Pool) -> Array

Compute SHAP feature attributions for each sample in pool.

Returns:

  • Regression / Binary: Matrix{Float64} of shape (n_samples, n_features).
  • Multiclass: Array{Float64,3} of shape (n_samples, n_features, n_classes).

Each row sums exactly to raw_prediction - E[raw_prediction], where the expectation is taken under a uniform distribution over tree leaves.

shap_values(model, data; cat_features=nothing) -> Array

Compute SHAP feature attributions for each sample.

  • Regression / Binary: returns Matrix{Float64} of shape (n_samples, n_features).
  • Multiclass: returns Array{Float64,3} of shape (n_samples, n_features, n_classes).

Each row sums approximately to prediction - mean_prediction.

Example

shap = shap_values(model, X_test)
# shap[i, j] is the contribution of feature j to sample i's prediction

Model Persistence

MichiBoost.save_model — Function
save_model(model, filepath::AbstractString)

Serialize a trained model to disk using Julia's Serialization module.

Works with both wrapper types (MichiBoostRegressor, MichiBoostClassifier) and raw MichiBoostModel objects.

See also load_model.

MichiBoost.load_model — Function
load_model(filepath::AbstractString) -> MichiBoostModel

Load a model previously saved with save_model.

Example

save_model(model, "my_model.jls")
loaded = load_model("my_model.jls")
predict(loaded, X_test)  # works with the raw MichiBoostModel
